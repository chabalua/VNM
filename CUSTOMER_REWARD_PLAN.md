## 4. DATA FLOW: Từ Đơn Hàng → Tính Thưởng KH

```
Đặt hàng → Chọn KH → Tạo đơn
       ↓
  orders.json (lưu khMa + items với nhom)
       ↓
  cusAggregateOrdersMonth(khMa, monthKey)
       ↓
  Tự động tổng hợp:
    • dsNhomC = sum(afterKM) where nhom='C'
    • dsNhomDE = sum(afterKM) where nhom='D'
    • dsVipN1 = sum(afterKM) where classifyVIP='N1'
    • dsVipN2 = sum(afterKM) where classifyVIP='N2'  
    • dsSBPS = sum(afterKM) where isSBPS_TE=true
    • skuNhomD = countUnique(ma) where nhom='D'
       ↓
  cusGetMonthData(kh, monthKey)
    → Ưu tiên manual override nếu có
    → Fallback sang auto từ đơn hàng
       ↓
  calcTotalReward(kh, monthData)
    → calcVNMShopReward()   → thưởng nhóm C
    → calcVIPShopReward()   → thưởng nhóm DE
    → calcSBPSReward()      → thưởng SBPS
       ↓
  Hiển thị: Card KH + Dashboard + Quick-view khi đặt hàng
```

## 5. PHÂN LOẠI SP CHI TIẾT (cho code)

### 5.1. Nhận diện SBPS_TE (Sữa bột pha sẵn trẻ em)

```javascript
function isSBPS_TE(product) {
  // Mã bắt đầu bằng 02H (SBPS)
  // VD: 02HO38 (Optimum), 02HL37 (Grow Plus), 02HG38 (Alpha Gold)
  return /^02H/.test(product.ma);
}
```

### 5.2. Nhận diện nhóm SBPS

```javascript
function classifySBPS(product) {
  var ma = product.ma;
  // N1: Dielac Gold, Dielac Grow, Optimum A2
  if (/^02HG/.test(ma) || /^02HD/.test(ma) || /^02HT/.test(ma)) return 'N1';
  // N2: Optimum Gold, Dielac Grow Plus
  if (/^02HO/.test(ma) || /^02HL/.test(ma) || /^02HA/.test(ma) || /^02HB/.test(ma)) return 'N2';
  // N3: Yoko, Optimum Colos
  if (/^02HY/.test(ma) || /^02HS/.test(ma)) return 'N3';
  return null; // Không thuộc SBPS
}
```

### 5.3. Nhận diện VIP Shop N1 vs N2

```javascript
function classifyVIPProduct(product) {
  var ma = product.ma;
  var ten = (product.ten || '').toLowerCase();
  
  // EXCLUDED — không áp dụng VIP Shop
  if (/^(07TA6|07TC6|07TR6|10BT0|07SN1|07SR1|14SA|14SD|14SC|10VS0|10VA0)/.test(ma)) 
    return 'excluded';
  
  // Chỉ áp dụng cho nhóm D + E (SCA, SCU, Phô mai, NGK)
  if (product.nhom !== 'D') return 'excluded'; // Tạm coi nhóm E nằm trong nhóm khác
  
  // N1 — Chủ lực
  // SCA Trắng CĐ, SCA Nha đam CĐ, SCA Star
  if (/^(07TR|07KD)/.test(ma) && /có đường|cd/i.test(ten)) return 'N1';
  if (/^07SR11/.test(ma)) return 'N1'; // Star
  // Probi CĐ 65ml, 130ml
  if (/^07UR13/.test(ma)) return 'N1'; // Probi truyền thống (CĐ)
  // Phô Mai
  if (/phô mai|pho mai/i.test(ten) && !/thạch|que/i.test(ten)) return 'N1';
  
  // N2 — Tập trung, Tiềm năng
  // SCA D các loại (trừ CĐ), SCA Green Farm, SCU Probi (trừ CĐ), SCU Green Farm
  if (/^07/.test(ma)) return 'N2'; // Mặc định SCA/SCU = N2
  if (/^06/.test(ma)) return 'N2'; // SCU Susu/Hero
  
  // Nước dừa, Kombucha = N2
  if (/^14GC|^14GU|^14GK/.test(ma)) return 'N2';
  
  return 'N2'; // Default D = N2
}
```
